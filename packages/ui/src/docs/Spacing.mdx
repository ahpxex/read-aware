# Spacing

ReadAware uses Tailwind's default spacing scale. The rhythm is generous -- the
interface breathes.

## Common spacing values

| Token | Value | Usage |
|-------|-------|-------|
| `gap-2` | 0.5rem (8px) | Tight grouping (label to input) |
| `gap-4` | 1rem (16px) | Default component spacing |
| `gap-6` | 1.5rem (24px) | Section padding, nav gaps |
| `gap-8` | 2rem (32px) | Between content blocks |
| `gap-10` | 2.5rem (40px) | Grid gaps in definition lists |

## Guidelines

- Prefer larger spacing. When in doubt, add more whitespace.
- Use the `Stack` component with `gap` prop for consistent vertical/horizontal rhythm.
- ReadAware ships as a desktop app shell: the window is a fixed frame and every
  surface (shelf, reader, panels) scrolls internally. Padding belongs to those
  surfaces, not to a scrolling document -- there is no responsive page-padding
  ladder.
- Avoid custom spacing values. The Tailwind scale provides enough granularity.
