# Colors

RadAware uses a deliberately limited palette. The foundation is a warm paper tone
with Tailwind's `stone` scale for text and UI elements.

## Border

The standard border color is `border-border` which resolves to `rgb(28 25 23 / 0.1)` --
stone-900 at 10% opacity. This keeps dividers visible but quiet.

## Usage guidelines

- **Backgrounds:** Use `bg-paper` for primary surfaces. Use `bg-paper-warm` sparingly
  for subtle differentiation.
- **Text:** `stone-950` for primary text, `stone-700` for body copy, `stone-500` for
  secondary labels and captions.
- **No accent colors.** The palette is intentionally monochrome. Color, if ever
  introduced, should be used with extreme restraint.
